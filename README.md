# AI-Resume-Analyzer-Job-Match-Platform-
AI Resume Analyzer &amp; Job Match Platform is a full-stack AI-based web application that helps users analyze resumes, improve ATS scores, compare resumes with job descriptions, identify missing skills, and receive AI-generated career recommendations through interactive visualizations and NLP techniques.

#TechStack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL
AI/NLP: OpenAI API, Gemini API, NLP
Visualization: Recharts / Chart.js
