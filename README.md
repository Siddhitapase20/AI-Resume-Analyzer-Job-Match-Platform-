# AI Resume Analyzer & Job Match Platform

An AI-powered resume analysis platform that extracts resume content, analyzes ATS compatibility, matches resumes with job descriptions, detects missing skills, provides AI-based improvement suggestions, and recommends suitable career roles using NLP and modern web technologies.

## Tech Stack
- React.js
- Node.js
- Express.js
- NLP
- OpenAI API
- PostgreSQL

## Features
- Resume Upload
- ATS Score Analysis
- Job Description Matching
- Skill Extraction
- AI Suggestions
- Job Recommendations
- Visual Dashboard
